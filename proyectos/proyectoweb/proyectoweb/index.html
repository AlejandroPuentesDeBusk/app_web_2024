Sobre Mí

Mi nombre es Alejandro Puentes De Busk soy alumno de la Universidad Tecnológica de Durango, actualmente estudio la carrera de Tecnologías de la Información.

Las tecnologías de desarrollo web que actualmente utilizamos en los proyectos que se desarrollan son: HTML para la maquetación, CSS hojas de estilo para el diseño, JavaScript programación del lado del cliente, Django como programación del lado del servidor y como Gestor de Base de Datos MariaDB o MySQL.

Te puedes poner en contacto conmigo por e-mail: puentesalejandro11@gmail.com
